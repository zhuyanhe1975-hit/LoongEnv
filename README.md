# LoongEnv

LoongEnv 是团队共享中枢仓，负责沉淀跨成员可见的文档、导航、部署入口、诊断结论与 AI 协作入口。

它**不替代成员个人工作仓**：成员仓负责实现与实验，中枢仓负责共享、对齐、回流与可见性。
