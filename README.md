# LoongEnv

LoongEnv 是团队开发中枢仓，用于汇聚共享文档、设计、部署信息、诊断结果以及 AI 成员协作约定。

这个仓库不替代成员个人工作仓，而是负责共享、对齐、回流和协作可见性。
