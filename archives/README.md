# archives

`archives/` 是 `LoongEnv` 中枢仓的**共享归档命名空间**。

这里用于承载跨成员复用、可长期沉淀、需要在中枢仓内统一索引的共享结果，而不是成员仓中的实现代码。

- [deploy](./deploy/README.md)
- [diagnose](./diagnose/README.md)

## 模板

- [部署归档模板](./deploy/_template.md)
- [诊断归档模板](./diagnose/_template.md)

## 说明

- `archives/` 表达的是**中枢仓归档语义**
- 它不等于项目正向设计架构 4D+C 的真实实现层
- 相关背景请查看 [4D+C 架构说明](../docs/4d-c-architecture.md)
