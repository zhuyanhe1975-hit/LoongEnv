# archives/deploy

这里是中枢仓中的**部署类共享归档区**。

它用于沉淀统一发布入口、共享部署说明、交付记录与团队级发布结论。

## 与旧目录的关系

- 原根目录 `deploy/` 已进入迁移态
- 后续新的共享部署沉淀应优先进入 `archives/deploy/`

## 模板

- [部署归档模板](./_template.md)

## 边界

- 这里归档共享部署结果，不承载成员仓内部流水线实现
- 4D+C 中的 `Deploy` 若属于项目实现结构，应保留在成员仓或项目架构文档中
