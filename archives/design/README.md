# archives/design

这里是中枢仓中的**设计类共享归档区**。

它用于沉淀需要跨成员复用的设计结论、方案草图、结构建议与阶段性设计文档。

## 与旧目录的关系

- 原根目录 `design/` 已进入迁移态
- 后续新的共享设计沉淀应优先进入 `archives/design/`

- [设计归档模板](./_template.md)

## 边界

- 这里归档共享结论，不承载成员仓内部实现
- 4D+C 中的 `Design` 若属于项目实现结构，应保留在成员仓或项目架构文档中
