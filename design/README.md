# design

> 本目录已进入迁移过渡期。

原先放在根目录 `design/` 下的共享设计归档，后续将逐步迁移到：

- [archives/design/](../archives/design/README.md)

## 为什么迁移

- `Design` 同时也是 4D+C 项目架构中的一个维度
- 若继续直接占据中枢仓根目录，容易被误读为项目模块目录
- `archives/design/` 更明确表达“中枢仓中的共享归档区”

## 当前规则

- 旧目录暂保留，用于兼容和迁移说明
- 新的共享设计沉淀请优先写入 `archives/design/`
